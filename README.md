# Smart Grid IDS Dashboard

## Overview

This project is a web-based dashboard designed to visualize anomaly detection in a smart grid system. It simulates real-time monitoring of power system parameters such as voltage, frequency, and phase data.

## Features

* Real-time visualization of PMU (Phasor Measurement Unit) data
* Detection of anomalies such as FDIA and replay attacks
* Interactive dashboard with graphs and alerts
* Simulated data for proof-of-concept demonstration

## Technologies Used

* HTML
* CSS
* JavaScript

## Demo

Live Website: https://yourusername.github.io/smart-grid-ids-dashboard

## Purpose

This project was developed as part of a cybersecurity-focused smart grid intrusion detection system. It demonstrates how visualization and anomaly detection can enhance grid security.

## Note

This is a prototype using simulated data and does not connect to a real smart grid system.
